# MangaReader

Легкий офлайн-ридер манги на SwiftUI для iOS с поддержкой внешних папок (Files/iCloud), автосканированием глав и сохранением прогресса чтения.

## Возможности

- Подключение нескольких библиотек через выбор папки в Files.
- Автоматическое сканирование структуры:
	- `manga/chapter/images`
	- `manga/volume/chapter/images`
	- или выбранная папка как один тайтл.
- Режимы чтения:
	- вертикальный
	- слева направо
	- справа налево
	- автоопределение режима по страницам главы.
- Запоминание точки чтения (глава, страница, режим, настройки отображения).
- Синхронизация прогресса через файл `MangaReaderProgress.json` в активной папке библиотеки.
- Сортировка тайтлов (по прогрессу, дате чтения, размеру, названию).
- Поддержка форматов: `jpg`, `jpeg`, `png`, `webp`, `bmp`, `gif`, `tiff`, `heic`, `avif`.

## Как пользоваться

1. Откройте проект `MangaReader.xcodeproj` в Xcode.
2. Запустите приложение на iPhone/iPad (Simulator или устройство).
3. В приложении нажмите `Добавить библиотеку` и выберите папку с мангой.
4. Откройте тайтл, выберите главу и читайте.

## Структура проекта

- `MangaReader/ContentView.swift` - экран библиотеки, выбор источника, настройки.
- `MangaReader/MangaLibraryViewModel.swift` - загрузка/сканирование библиотек, сортировка, доступ к папкам.
- `MangaReader/MangaScanner.swift` - поиск тайтлов, глав и изображений на диске.
- `MangaReader/ReaderView.swift` - UI чтения, режимы, восстановление позиции.
- `MangaReader/MangaReadingStateStore.swift` - хранение и синхронизация прогресса.
- `MangaReader/MangaModels.swift` - модели библиотеки, глав и состояния чтения.

## Ограничения

- Для внешних папок нужен доступ через security-scoped bookmarks; если доступ потерян, библиотеку нужно выбрать заново.
- Приложение рассчитано на офлайн-чтение локальных/icloud-папок, без встроенного скачивания манги из сети.
